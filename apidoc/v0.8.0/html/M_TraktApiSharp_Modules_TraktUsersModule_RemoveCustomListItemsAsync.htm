TraktUsersModuleRemoveCustomListItemsAsync Method
Removes items from an user's custom list. Accepts shows, seasons, episodes, movies and people.

OAuth authorization required.

See "Trakt API Doc - Users: Remove List Items" for more information.

It is recommended to use the TraktUserCustomListItemsPostBuilder to create an instance of the required TraktUserCustomListItemsPost. See also .
