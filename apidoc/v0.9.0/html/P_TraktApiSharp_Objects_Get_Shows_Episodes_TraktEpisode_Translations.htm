Property Value

Type: IEnumerableTraktEpisodeTranslation
Remarks
This property is set automatically if this episode is in a Episodes collection and the episode's season is in a collection of seasons returned by GetAllSeasonsAsync(String, TraktExtendedInfo, String) and a translation language code was specified. This property is also set automatically if this episode is in a collection returned by GetSeasonAsync(String, Int32, TraktExtendedInfo, String) and a translation language code was specified.
