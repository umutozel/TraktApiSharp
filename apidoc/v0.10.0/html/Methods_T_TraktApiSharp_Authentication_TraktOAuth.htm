GetAuthorizationAsync
Exchanges the OAuth authorization code from the user for a new access token. Uses the current OAuthAuthorizationCode for the request, which has to be set before a call to this method. Also uses the current ClientId, ClientSecret and RedirectUri for the request. Assigns the returned TraktAuthorization instance to Authorization, if successful.

See "Trakt API Doc - OAuth: Get Token" for more information.
