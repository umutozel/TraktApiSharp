TraktShowFilter Class
Provides additional filter parameters for some TraktShowsModule methods.

Supported by GetMostAnticipatedShowsAsync(TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32), GetMostCollectedShowsAsync(TraktTimePeriod, TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32), GetMostPlayedShowsAsync(TraktTimePeriod, TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32), GetMostWatchedShowsAsync(TraktTimePeriod, TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32), GetPopularShowsAsync(TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32), GetRecentlyUpdatedShowsAsync(NullableDateTime, TraktExtendedInfo, NullableInt32, NullableInt32) and GetTrendingShowsAsync(TraktExtendedInfo, TraktShowFilter, NullableInt32, NullableInt32).

This class has an fluent interface.

See "Trakt API Doc - Filters" for more information.
