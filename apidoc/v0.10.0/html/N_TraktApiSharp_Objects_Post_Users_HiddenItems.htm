TraktUserHiddenItemsPost
A Trakt hidden items post, containing all movies, shows and / or seasons, which should be added to the user's hidden items.
TraktUserHiddenItemsPostBuilder
This is a helper class to build a TraktUserHiddenItemsPost.

It is recommended to use this class to build a hidden items post.

An instance of this class can be obtained with Builder(TraktHiddenItemsSection).

TraktUserHiddenItemsPostMovie
A Trakt hidden items post movie, containing the required movie ids.
TraktUserHiddenItemsPostSeason
A Trakt hidden items post season, containing the required season ids.

Can also contain optional seasons.

TraktUserHiddenItemsPostShow
A Trakt hidden items post show, containing the required show ids.

Can also contain optional seasons.

TraktUserHiddenItemsPostShowSeason
A Trakt hidden items post season, containing the required season number.
