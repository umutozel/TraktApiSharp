RevokeAuthorizationAsync Method (String, String)
TraktAuthenticationRevokeAuthorizationAsync Method (String, String)
Revokes the given access token. If, successful, the given access token will be invalid and the user has to be re-authenticated, e.g. by calling GetAuthorizationAsync.

See "Trakt API Doc - OAuth: Revoke Token" for more information.
