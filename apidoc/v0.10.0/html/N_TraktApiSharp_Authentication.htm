TraktApiSharp.Authentication Namespace
Contains the authentication modules for OAuth- and Device-Authentication and classes for holding authorization information.
Classes
  ClassDescription
TraktAuthentication
Provides access to authentication methods common to both OAuth and Device authentication, such as refreshing and revoking the current access token.

Besides, contains current authorization state information.

TraktAuthorization
Represents a Trakt authorization response, which contains information, such as access token and refresh token.

See also , , , , , , , , , , , , and .

See "Trakt API Doc - OAuth: Get Token" and "Trakt API Doc - Device: Get Token" for more information.

TraktDevice
Represents a Trakt device response.

See also , .

See "Trakt API Doc - Devices: Device Code" for more information.

TraktDeviceAuth
Provides access to Device authentication methods, such as creating a new device and polling for an access token.
TraktOAuth
Provides access to OAuth authentication methods, such as creating a new authorization URL and getting a new access token.
