Exchanges the current refresh token for a new access token, without re-authenticating the associated user. Uses the current ClientSecret and RedirectUrifor the request. Assigns the returned TraktAuthorization instance to Authorization, if successful.

See "Trakt API Doc - OAuth: Get Token" for more information.
