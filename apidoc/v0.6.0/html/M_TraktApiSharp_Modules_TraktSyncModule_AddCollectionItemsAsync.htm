Adds items to the user's collection. Accepts shows, seasons, episodes and movies.

OAuth authorization required.

See "Trakt API Doc - Sync: Add to Collection" for more information.

It is recommended to use the TraktSyncCollectionPostBuilder to create an instance of the required TraktSyncCollectionPost. See also .
