TraktScrobbleModulePauseEpisodeWithShowAsync Method
Pauses watching a TraktEpisode in a media center. Use this method, if the given episode has no valid ids.

OAuth authorization required.

See "Trakt API Doc - Scrobble: Start" for more information.
