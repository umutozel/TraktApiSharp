The Trakt show, which will be added.

A rating from 1 to 10 for the given show.
ratedAt
Type: SystemDateTime
The datetime, when the given show was rated. Will be converted to the Trakt UTC-datetime and -format.
seasons
Type: TraktApiSharp.Objects.PostPostRatingsSeasons
An PostRatingsSeasons instance, containing season and episode numbers.

If it contains episode numbers, only the episodes with the given episode numbers will be added to the ratings.

Return Value

Type: TraktSyncRatingsPostBuilder
The current TraktSyncRatingsPostBuilder instance.
Exceptions
ExceptionCondition
ArgumentNullException Thrown, if the given show is null. Thrown, if the given show ids are null.
ArgumentException Thrown, if the given show has no valid ids set. Thrown, if the given show has an year set, which has more or less than four digits.
ArgumentOutOfRangeException Thrown, if at least one of the given season numbers in seasons is below zero. Thrown, if at least one of the given episode numbers in seasons is below zero. Thrown, if the given rating is not between 1 and 10.
