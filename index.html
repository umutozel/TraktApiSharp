TraktApiSharp

TraktApiSharp is a .NET library, written in CSharp, which makes it easy to use the Trakt.tv API.

Download Source Code

Open Source and free under the MIT License

Supported Platforms

.Net Framework >= 4.5 - ASP.NET Core 1.0 - Windows 8 / 8.1 / 10 / UWP

Xamarin Android - Xamarin iOS - Windows Phone 8.1

Full Trakt.tv API Coverage

You can use every available and documented Trakt.tv API request.

Authentication Support

The library makes it easy to authenticate your users either with OAuth- or with Device-Authentication.

Completely asynchronous

Every API request is executed asynchronously and does not block your application UI.

API Environments

Both production and sandbox environment of the Trakt.tv API are supported.

Serialization Service

Get the user's current OAuth authorization as JSON with the builtin serialization service and save it for later usage.

Language Service

The builtin language service makes it easy to lookup the language and country names for language and country codes returned by the Trakt.tv API.
