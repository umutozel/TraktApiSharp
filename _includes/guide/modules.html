Modules

The modules in the library represent each a section in the official Trakt API documentation.
You can not create a new instance of any module. The only way to access a module is through a TraktClient instance.


Authentication

This module contains all methods, that are common to both Trakt API Authentication - OAuth and Trakt API Authentication - Devices sections, such as refreshing and revoking the current authorization.
Access this module via TraktClient.Authentication.


Authentication - OAuth

This module contains all methods, that are described in the Trakt API Authentication - OAuth section.
Access this module via TraktClient.OAuth.


Authentication - Devices

This module contains all methods, that are described in the Trakt API Authentication - Devices section.
Access this module via TraktClient.DeviceAuth.
